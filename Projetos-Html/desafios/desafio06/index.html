Desafio Tags em HTML

Esse exercício faz uma lista de coisas que podem ser adicionadas em uma página. O desafio aqui é arrumar uma maneira que, ao passar o mouse sobre um termo, mostre qual a tag utilizada para fazer isso.

  • Págrafo
  • Títulos
  • Imagens
  • Abreviações
  • Lista numeradas
  • Lista com marcadores
  • Sub-título
  • Icon
  • Negrito
  • Itálico
  • Links
